<h1 align="center">恶魔轮盘赌</h1>

## 项目介绍

vue + ts

## 玩法介绍

游戏大体玩法遵循[恶魔轮盘赌](https://mikeklubnika.itch.io/buckshot-roulette)，采用双人本地对战的方式。  
为了平衡先后手的影响。后手的玩家会在第一轮多获得一个道具

## 启动方式

### 线上游玩

[线上地址](https://buckshot-roulette-one.vercel.app/)

### 本地游玩

拉去项目后

```
$ pnpm install
$ pnpm dev
```

即可在本地启动项目
